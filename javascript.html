Understanding JavaScript (JS)

JavaScript (JS) is a versatile, high-level programming language that enables you to create dynamic and interactive content on websites. It allows you to implement complex features such as interactive forms, animations, and real-time updates without refreshing the page.

What is JavaScript?

JavaScript is a programming language that runs in web browsers and is used to create dynamic web pages. It can interact with HTML and CSS to manipulate the structure, style, and behavior of a webpage. It is one of the core technologies of the web, alongside HTML and CSS.

Embedding JavaScript in HTML

JavaScript can be embedded in an HTML document in two main ways:

  • Inline JavaScript: Directly within HTML elements using the onclick, onmouseover, etc., attributes.
  • External JavaScript: By linking to an external .js file using the <script> tag.

JavaScript Syntax

JavaScript syntax consists of statements, which are executed in order, and are made up of expressions (values and operators). Here's an example:

let name = "John";
console.log(name);

This example defines a variable name and logs it to the console.

JavaScript Variables

Variables are used to store data that can be used later in the code. There are three ways to declare variables in JavaScript:

  • var: A legacy keyword, generally avoided in modern JS.
  • let: Used to declare block-scoped variables.
  • const: Used for variables whose value cannot be changed.

Example of JavaScript Variables

let age = 25;
const birthYear = 1995;
console.log(age);

JavaScript Functions

Functions in JavaScript are blocks of code designed to perform a particular task. A function is defined using the function keyword.

function greet(name) {
   console.log("Hello, " + name + "!");
}
greet("John");

The function greet() is called with the argument "John", which outputs: Hello, John!.

JavaScript Loops

Loops are used to repeat a block of code multiple times. Here’s an example using a for loop:

for (let i = 0; i < 5; i++) {
   console.log(i);
}

This loop prints numbers from 0 to 4 in the console.

JavaScript Conditional Statements

Conditional statements are used to perform different actions based on different conditions. Here’s an example:

let age = 20;
if (age >= 18) {
   console.log("You are an adult.");
} else {
   console.log("You are a minor.");
}

This code checks if a person is 18 or older and logs an appropriate message.

Conclusion

JavaScript is a powerful language used to create interactive and dynamic webpages. It enables features like user interaction, form validation, animations, and real-time content updates. With JavaScript, you can enhance the user experience and build complex web applications.

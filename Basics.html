Basics of HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses a system of tags to structure content.

HTML Structure

Every HTML document starts with a declaration that defines the document type:

<!DOCTYPE html>

The structure consists of two main parts:

  • <head>: Contains metadata, title, and links to stylesheets.
  • <body>: Contains the visible content of the page.

Basic Tags

Some commonly used HTML tags:

  • <h1> to <h6>: Headings, with <h1> being the largest and <h6> the smallest.
  • <p>: Paragraph tag used for text.
  • <a>: Anchor tag used for hyperlinks.
  • <img>: Image tag used to embed images.
  • <ul>: Unordered list, and <ol>: Ordered list, with <li> for list items.

Adding an Image

To add an image, use the <img> tag. Here's an example:

<img src="image.jpg" alt="Description of Image">

Example Image:

The src attribute specifies the image file path, and the alt attribute provides a text description for accessibility.

Conclusion

HTML is the foundation of all web pages. It structures content and allows you to add multimedia, links, and text to a webpage.
